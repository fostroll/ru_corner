Оборудование для производства пенобетонных блоков
Продам завод по производству неавтоклавного пенобетона.
Производительность 31 м. куб. в смену (8 часов)
В стоимость завода входит:
Узел приготовления пенобетонной смеси:
1. Силос для цемента 55 т
2. Бункер для песка 2 т
3. Скиповый подъемник для загрузки песка
4. Шнековый питатель для цемента 3 м.
5. Ленточный питатель для подачи песка
6. Смеситель 0,55 м.куб (Санни-5)
7. Насосы для подачи воды
9. Компрессор 380 V
10. Пульт управления
11. Весовой дозатор на тензодатчиках
Узел формирования пеноблока:
1.Распалубочный комплекс Санни 1УКЖД-Ф2Р-9 (56 форм)
2. Автоматическое распалубочное устройство САННИ АРУ- Н
3. 56 неразборная высокоточная форма со съемным дном для стенового блока 588х288х200
4. Ж/д рельсы с замкнутой круговой системой. Автоматическое движение.
5. Электротельфер г/п 1,5 т
6. Гидравлическая маслостанция
7. Пульт управления8. Захват для распалубки форм "Санни ЗДР"
9. Вилочный погрузчик.
10.Гидравлическая тележка рохля
Мобильная установка пенобетона для выездных работ:
1. Смеситель Санни-16ПГ.
Линию по стеновым блокам можно увеличить. Для работы на данном оборудовании требуется 3-5 человек, все операции автоматизированы, в производстве не требуется тяжелых физических усилий.
Есть готовая база поставщиков качественного сырья.
Передадим готовые и отлаженные рецептуры производства пенобетона по различным маркам плотности.

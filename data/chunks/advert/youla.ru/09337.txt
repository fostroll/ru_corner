Dyson Supersonic - фен для волос, продуманный иначе.
Быстрое высушивание волос
Высушивание волос не должно длиться вечно. Для быстрого высушивания волос необходим интенсивный, управляемый поток воздуха.
Данная модель помогает избежать чрезмерного нагревания волос, защищая их естественный блеск. Температура воздуха замеряется 20 раз в секунду, постоянно находясь под контролем.
Благодаря технологии Heat Shield поверхность насадки не нагревается. Даже при близком контакте с волосами.
Только фен для волос Dyson Supersonic поставляется в комплекте с магнитными насадками, которые легко устанавливаются.
Насадка-концентратор для укладки волос позволяет формировать каждый локон в отдельности, не затрагивая другие влажные волосы.
Диффузор Dyson равномерно рассеивает воздух вокруг локонов, позволяя выделить каждый завиток.
Насадка Dyson для создания направленного потока бережно высушивает волосы с помощью равномерного, объемного потока воздуха.
3 точные настройки скорости: быстрое высушивание волос, стандартный режим и укладка. 4 уровня нагрева: 100°C - быстрое высушивание волос и укладка, 80°C - стандартное высушивание волос, 60°C - бережное высушивание волос, 30°C - поток прохладного воздуха.

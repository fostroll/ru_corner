Zoom передумал вводить сквозное шифрование только для платных пользователей
Сервис для видео-конференций Zoom решил ввести сквозное шифрование звонков для всех пользователей. Об этом сообщается на сайте компании.
Как сказано в релизе, теперь такой тип шифрования будет доступен не только для платных пользователей, но и для бесплатных. Решение было принято после сотрудничества с организациями по гражданским свободам, экспертами по шифрованию и представителями правительства, отмечает компания.
Для того, чтобы воспользоваться сквозным шифрованием, пользователь должен будет пройти дополнительную верификацию номера мобильного телефона, сказано в сообщении.
Ранее Zoom объяснил отказ от сквозного шифрования для бесплатных пользователей необходимостью сотрудничать с ФБР.
